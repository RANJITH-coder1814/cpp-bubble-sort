# cpp-bubble-sort
This project demonstrates the Bubble Sort algorithm using C++. Bubble Sort is a simple sorting technique that repeatedly compares adjacent elements and swaps them if they are in the wrong order.

## 📌 Features

* Beginner-friendly C++ implementation
* Takes user input for array elements
* Sorts numbers in ascending order
* Demonstrates a basic sorting algorithm

## 🛠️ Technologies Used

* C++
* Standard Library (`iostream`)

## ▶️ Example

Input

```
Enter number of elements: 5
Enter elements:
5 3 8 4 2
```

Output

```
Sorted array:
2 3 4 5 8
```

## 📚 Algorithm

1. Compare adjacent elements in the array.
2. Swap them if they are in the wrong order.
3. Repeat the process for all elements.
4. Continue until the array is completely sorted.

## ⏱️ Time Complexity

* Best Case: **O(n)**
* Average Case: **O(n²)**
* Worst Case: **O(n²)**
